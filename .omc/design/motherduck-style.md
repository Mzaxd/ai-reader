# MotherDuck 风格整理（基于官网当前实现）

更新日期：2026-02-17  
参考来源：`https://motherduck.com/` 首页实际样式与内联 CSS

## 1) 视觉方向
- 关键词：Playful Data Product、Cartoon Tech、粗描边 + 高对比 + 轻拟物
- 不是“柔和渐变 SaaS”，而是“明亮底色 + 黑色描边 + 高饱和强调色”
- 通过插画、滚动字带、轻量上浮动效建立记忆点

## 2) 颜色骨架（实测高频）
- 轮廓/正文主色：`#383838`
- 主背景白：`#FFFFFF`
- 纸张浅底：`#F8F8F7`
- 奶油底色：`#F4EFEA`
- 主品牌蓝：`#6FC2FF`
- 主品牌黄：`#FFDE00`
- 强调橙：`#FF9538`
- 辅助高亮蓝：`#2BA5FF`

## 3) 字体与层级
- UI 主体：`Inter`
- 功能/标签/滚动字带：`Aeonik Mono`（原站），可用 `IBM Plex Mono / JetBrains Mono` 近似
- 标题倾向：全大写、较紧凑、粗字重
- 正文倾向：16-18px，行高 1.55-1.75，保证可读

## 4) 组件气质
- 边框：高频 `2px solid #383838`
- 圆角：小圆角（2px~12px），不是大面积胶囊
- 按钮：高对比填充 + 实线描边 + 明确悬停位移
- 卡片：平面色块 + 边框 + 轻阴影（强调“纸片层叠”）

## 5) 动效语言
- Marquee（滚动信息带）是标志性元素
- 入场：`opacity + translateY`，幅度小、节奏快（120ms~300ms）
- 漂浮：插画元素 3-6s 慢周期上下浮动
- 交互反馈：hover 轻抬升（1-3px）+ 阴影增强
- 必须支持 `prefers-reduced-motion`

## 6) 页面结构模式
- 顶部常见“告警/活动条”+ 主导航
- Hero 常见“左文字 + 右插画/场景”
- 中部穿插高对比色条与滚动文本带
- 区块之间用明显分割线或色块切层

## 7) 映射到 AI Reader 的落地规则
- 阅读器基调：`#F8F8F7` 纸面感 + `#383838` 信息骨架
- 关键功能（AI问答/人物关系/时间线/事件）用蓝黄橙做语义分区
- 交互优先级：
  1. 阅读连续性（目录、进度、书签）
  2. AI 辅助可见性（右侧工作台常驻）
  3. 关键动作显性（导入、继续阅读、提问）

## 8) 避免事项
- 避免泛紫色 SaaS 渐变
- 避免过度玻璃拟态
- 避免“全圆角 + 低对比”导致信息发虚
- 避免无语义动效（只保留有目的的动效）
