# MotherDuck 视觉参考（快速版）

更新时间：2026-02-17

## 直接可复用的视觉特征
- 主体基色：白/浅奶油底（`#FFFFFF`, `#F8F8F7`, `#F4EFEA`）
- 主强调：天空蓝（`#6FC2FF`）
- 高关注条带：亮黄（`#FFDE00`）
- 二级强调：橙色（`#FF9538`）
- 统一轮廓线：深灰（`#383838`）

## 字体策略
- UI 和按钮：等宽/近等宽风格（官方常见 Aeonik Mono）
- 正文和说明：Inter
- 中文落地：Inter + PingFang / Noto Sans CJK

## 动效策略
- 滚动字带（marquee）
- 元素入场上移（fade + translateY）
- 插画/云朵慢速漂浮（3-6s）
- hover 轻抬升 + 阴影加强

## 版式策略
- 顶部活动条 + 导航
- Hero 左文案右插画
- 强分割感区块（描边、色块、信息条）
- 卡片和按钮多为 2px 轮廓，不做重玻璃化

## AI Reader 映射建议
- 书库页：高对比入口（导入 / 继续阅读）+ 书卡进度
- 阅读页：左目录、中正文、右 AI 工作台并行
- 右侧工作台固定四标签：问答 / 人物 / 时间线 / 事件
- 所有 AI 输出都保留证据引用区块
